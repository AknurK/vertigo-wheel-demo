# Vertigo Wheel Demo

## Unity Version
Unity 2021.3.45f2

## How to Run
1. Open the project in Unity Hub
2. Open scene: `Scenes/SampleScene`
3. Press Play

## Project Description
This project demonstrates a spin wheel reward system implemented in Unity.
The wheel randomly selects rewards and applies game logic based on zone rules.
